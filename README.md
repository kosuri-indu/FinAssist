# FinAssist 

FinAssist is a complete personal finance management system that combines a simple, intuitive user interface with powerful AI analysis. It tracks your spending, reminds you about bills, provides intelligent insights, predicts future expenses, and answers your financial questions through a conversational AI. All your data stays secure in a PostgreSQL database with robust authentication. The system intelligently uses semantic caching and rate limiting to work efficiently within the Groq free tier limits. Whether you want to understand where your money goes, save more, or never miss a bill payment again, FinAssist provides the tools you need.
